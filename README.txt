Set Up
---------------------------------------------------------------------------------------------------------
> after hosting, turn on the live mode in facebook developer, entering whitelist server IP and website url
  for OAuth callbacks

> Turn on the comments of //href: 'https://localhost:44303/CallOut/SharePage?lat='+lat+'&lng='+lng,
  and delete or turn on comments of href: 'https://developers.facebook.com/docs/?lat=' + lat + '&lng=' + lng,
  in index page and ShareProblem pages.
  note - it is marked with //share details of each problems section in ShareProblem page"

> change those local host to new hosted name

> change the image URL in meta tag in Share Page

> change the URL of the project

> connect the database files
